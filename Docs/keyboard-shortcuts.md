# Keyboard Shortcuts Reference

Complete list of keyboard shortcuts available in the BlazorRTE Rich Text Editor.

> **Platform Note:** Replace `Ctrl` with `Cmd` (⌘) on macOS.

---

## Quick Reference Table

| Category | Shortcut | Action |
|----------|----------|--------|
| **Text Formatting** | | |
| | `Ctrl+B` | **Bold** |
| | `Ctrl+I` | *Italic* |
| | `Ctrl+U` | <u>Underline</u> |
| | `Ctrl+Shift+X` | ~~Strikethrough~~ |
| | `Ctrl+=` | Subscript (x₂) |
| | `Ctrl+Shift+=` | Superscript (x²) |
| **History** | | |
| | `Ctrl+Z` | Undo |
| | `Ctrl+Y` | Redo |
| | `Ctrl+Shift+Z` | Redo (alternative) |
| **Lists** | | |
| | `Ctrl+Shift+8` | • Bullet List |
| | `Ctrl+Shift+7` | 1. Numbered List |
| **Indentation** | | |
| | `Ctrl+]` | Increase Indent |
| | `Ctrl+[` | Decrease Indent |
| **Alignment** | | |
| | `Ctrl+L` | Align Left |
| | `Ctrl+E` | Align Center |
| | `Ctrl+R` | Align Right |
| | `Ctrl+J` | Justify |
| **Headings** | | |
| | `Ctrl+Alt+0` | Normal Text (¶) |
| | `Ctrl+Alt+1` | Heading 1 |
| | `Ctrl+Alt+2` | Heading 2 |
| | `Ctrl+Alt+3` | Heading 3 |
| **Font Size** | | |
| | `Ctrl+Shift+>` | Increase Size |
| | `Ctrl+Shift+<` | Decrease Size |
| **Insert** | | |
| | `Ctrl+K` | Insert Link |
| | `Ctrl+Enter` | Horizontal Rule |
| **Utility** | | |
| | `Ctrl+\` | Clear All Formatting |

- Screen reader compatible
- Voice control compatible (Dragon, Windows Speech Recognition)
- Keyboard-only navigation supported
